How to play

The objective to the game is up to you. We provide with a simulator of an epidemic in a city.

The game will end if all infections are discharged, or if all the population is infected.

Disclaimer

This game is inspired by the 2019-2020 SARS-CoV-2 epidemic, but it's not a reallistic recreation of it.

This is intended for entertainment purposes and the simulations in this game are not based in scientific research.

We take no responsiblity on simulations run within this game.

We respect all the victims of this pandemic, for which we wanted to show support by making a game that shows how economics and politics can impact negatively on the evolution of a pandemic.
